## 🚀 Como Executar o Projeto

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/SEU-USUARIO/mini-sistema-web.git
```

### 2️⃣ Copiar para o XAMPP

```text
/opt/lampp/htdocs/projeto
```

### 3️⃣ Iniciar o XAMPP

```bash
sudo /opt/lampp/lampp start
```

### 4️⃣ Criar o banco

```sql
CREATE DATABASE sistema_web;
```

### 5️⃣ Importar o banco

Importe o arquivo:

```text
database/banco.sql
```

### 6️⃣ Acessar o sistema

```text
http://localhost/projeto
```
